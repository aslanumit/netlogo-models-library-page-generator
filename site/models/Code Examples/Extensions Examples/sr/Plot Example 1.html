Plot Example 1

Run on NetLogoWeb

WHAT IS IT?

This model demonstrates how to use the R function plot to create a scatter plot and a line plot in an R plotting environment.

HOW IT WORKS

To create a scatter plot, the model uses the positions of the turtles. To create a line plot, the model uses the spatstat package to calculate the L-function of a generated set of data.

HOW TO USE IT

To initialize the R environment and create the turtles, click the setup button. Then, to create a scatter plot, click the test-plot button. The plot will be displayed in a separate R window. To create a line plot, click the test-plot2 button. The new plot will replace the plot in the existing R window.

EXTENDING THE MODEL

To extend the model, you can try setting the color of each point in the scatter plot to the color of the corresponding turtle.
