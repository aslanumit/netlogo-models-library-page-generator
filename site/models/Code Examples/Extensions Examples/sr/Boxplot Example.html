Boxplot Example

Run on NetLogoWeb

WHAT IS IT?

This model demonstrates the usage of the boxplot function in R, which is used to create box-and-whisker statistical visualizations.

HOW IT WORKS

The functionality of boxplot is demonstrated by plotting a set of random numbers with a normal distribution, created by NetLogo's random-normal primitive.

HOW TO USE IT

To initialize the R environment and open a display window, click the activate-javagd-plot-device button. Then, click either the test button or the test2 button to generate the data and plot it in the display window.

THINGS TO NOTICE

Notice that the mean and standard deviation specified in random-normal are reflected in the box-and-whisker plot, by the central horizontal bar and the height of the box, respectively.

THINGS TO TRY

Try changing the inputs to random-normal and see how it affects the plot.
