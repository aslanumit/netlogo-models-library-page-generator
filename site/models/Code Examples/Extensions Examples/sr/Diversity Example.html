Diversity Example

Run on NetLogoWeb

WHAT IS IT?

This model demonstrates how to use the simple R (sr) extensions to calculate a number of diversity metrics.

HOW IT WORKS

The model creates a bunch of random data for breeds of trees and calculates a number of diveristy metrics for each patch's trees using the simple R extension. The diversity metrics for each patch are then plotted in histograms

HOW TO USE IT

Just press setup and run.

THINGS TO NOTICE

Notice the similarities and differences between the different diversity metrics.

THINGS TO TRY

Try chaning the setup code to see how that effects the diversity histograms.
