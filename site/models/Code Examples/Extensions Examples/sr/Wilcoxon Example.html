Wilcoxon Example

Run on NetLogoWeb

WHAT IS IT?

This model demonstrates the usage of the wilcox.test function in R, which is used to create box-and-whisker statistical visualizations.

HOW IT WORKS

The functionality of wilcox.test is demonstrated by calculating it on lowland-bugs and highland-bugs and then outputin gand plotting the results.

HOW TO USE IT

This section could explain how to use the model, including a description of each of the items in the interface tab.

THINGS TO NOTICE

Click setup to initialize the model and then click go. Each time go is pressed, the turtles calculate their fitness and then the Wilcoxon test is run.

THINGS TO TRY

Try changing constants in the setup procedure and see what happens.
