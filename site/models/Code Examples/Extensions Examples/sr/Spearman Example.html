Spearman Example

Run on NetLogoWeb

WHAT IS IT?

This model demonstrates how to use the R function cor.test to test the correlation between two values, specifically using the Spearman method.

HOW IT WORKS

Each turtle is assigned a weight and height, which are generated randomly from the Weibull and Beta distributions, respectively. The model then uses the R function cor.test to test the correlation between the weight and height of the turtles, displaying the result in the Spearman's rho plot.

HOW TO USE IT

To initialize the R environment and create the turtles, click the setup button. Then, click the go button to run the model.

EXTENDING THE MODEL

To extend the model, you can try generating the weight and height values from other distributions, to see how that changes the result of the correlation test.
