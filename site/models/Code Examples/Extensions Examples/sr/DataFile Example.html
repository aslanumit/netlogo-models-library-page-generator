DataFile Example

Run on NetLogoWeb

WHAT IS IT?

This model demonstrates the use of the write.table and read.table functions in R, showing how they can be used to save and load turtle positions.

HOW IT WORKS

When writing to a file, the model creates a set of turtles and writes their positions to the specified file. When reading a file, the model loads the positions from the specified file, creating a new turtle for each position.

HOW TO USE IT

First, select your desired output file in the path input widget. You can type a path directly into the input field, or you can click the select path button to open a file chooser dialog. After selecting an output file, click the create-and-write-data button. To load the turtle positions back in, clear the world with clear-turtles, then click the load-and-show-data button.

For a more advanced writing example, you can use the write-data-with-vector-dataframe button to write the value of a turtle variable to the file along with each turtle's position. Note that the additional variable cannot be loaded after writing.

EXTENDING THE MODEL

Try extending the create-and-write-data and load-and-show-data procedures to allow writing and loading additional turtle variables, such as heading or color. For a more advanced extension, you can also try adding a procedure to load the data created by the write-data-with-vector-dataframe button.
