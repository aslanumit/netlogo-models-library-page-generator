Patches to R Matrix

Run on NetLogoWeb

WHAT IS IT?

This model demonstrates how to create an R matrix using the Simple R extension.

HOW IT WORKS

This model shows an example of how you can create an R matrix from the world's patch colors. The resulting matrix has the same dimensions as the world, and each element contains the corresponding patch's pcolor as a NetLogo number.

HOW TO USE IT

To initialize the R environment and set up the patches, click the setup button. Then, to create the R matrix, click send-patches-to-r. Finally, to display the resulting matrix, click create and show matrix.

THINGS TO TRY

Try changing the world dimensions to see how the resulting R matrix changes.

EXTENDING THE MODEL

To extend the model, you can try performing some sort of data analysis on the generated R matrix. For example, you could calculate the mean patch color. You can also try adding a procedure that reads the patch colors back in from the R matrix and sets the corresponding patch colors in the view.
