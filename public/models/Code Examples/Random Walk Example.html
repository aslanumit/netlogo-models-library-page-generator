Random Walk Example

Run on NetLogoWeb

WHAT IS IT?

This code example is a demo of a basic random walk. At each step, the yellow turtle changes its heading randomly.

THINGS TO NOTICE

The turtle's pen is down, so it leaves a trail behind it in the drawing.
