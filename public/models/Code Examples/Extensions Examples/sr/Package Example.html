Package Example

Run on NetLogoWeb

WHAT IS IT?

This model demonstrates the usage of additional R packages with the Simple R extension, providing an example with the spatstat package.

HOW IT WORKS

The model calculates the pair correlation function for 20 random turtle positions, using the R function pcf. The result is displayed in the pair correlation function plot.

HOW TO USE IT

Click the test-pcf1 and test-pcf2 buttons to see two different examples of the pair correlation function calculation using turtle positions.

EXTENDING THE MODEL

Try using some of the other functions from the spatstat package to analyze the turtle data. Alternatively, try importing another package and using its functions.
