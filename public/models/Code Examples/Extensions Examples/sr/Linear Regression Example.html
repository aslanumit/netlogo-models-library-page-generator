Linear Regression Example

Run on NetLogoWeb

WHAT IS IT?

This model demonstrates the use of the R function lm to fit a linear model to a dataset.

HOW IT WORKS

In this example, the turtles use the linear model fit by R to determine how far they should move each tick in order to get more food. Each turtle remembers how much food it found on each of the previous 20 days, which becomes the input to the linear model. Turtles that go without food for 20 days will die.

HOW TO USE IT

First, click the setup button to initialize the R environment and create the turtles. Then, click the go button to run the model.

EXTENDING THE MODEL

Try changing the create-food-sources procedure to distribute food differently, or to create multiple different types of food. You can also try changing the length of the memory variable to see how it affects which turtles live and which turtles die.
