RScript Example

Run on NetLogoWeb

WHAT IS IT?

This model demonstrates how to use the R function source to run an R script that is saved to an external file.

HOW IT WORKS

The model loads the R source from the specified file and runs it with the R function source, displaying the output in the output widget.

HOW TO USE IT

First, select your desired R script file in the path input widget. You can enter the path directly into the input field, or you can click the choose path... button to open a file chooser dialog. Then, click the test button, which will run your script and display the output in the output widget.

THINGS TO TRY

Try using your own R script instead of the one provided with NetLogo.
