Plot to File Example

Run on NetLogoWeb

WHAT IS IT?

This model demonstrates how to export an R plot to a file, both as an image and as a PDF.

HOW IT WORKS

The model uses the R function plot to create a scatter plot of the turtle positions, which is then exported to an image with the R function jpeg and to a PDF with the R function pdf.

HOW TO USE IT

First, select your desired output file name (without an extension!) in the plotfile input widget. You can either type directly into the input field or click the Choose file button, which will open a file chooser dialog. Then, click the test-plot button to initialize the R environment, create the turtles, and create and export the plot.

EXTENDING THE MODEL

To extend the model, you can try setting the color of each point in the scatter plot to the color of the corresponding turtle. You can also try exporting other types of plots to a file.
