Interactive Shell

Run on NetLogoWeb

WHAT IS IT?

This model demonstrates the use of the interactive shell provided with the Simple R extension.

HOW IT WORKS

When the interactive shell is set up, the model creates a set of turtles and assigns it to the local R variable ag.df. This variable can then be used in Simple R primitives such as sr:run, in addition to the interactive shell.

HOW TO USE IT

First click the setup button, which will create the turtles and set up the R environment. Then, click the open-interactiveshell button, or select the "Pop-out Interpreter" option from the "SimpleR Extension" menu. Start by typing the commands indicated in the note widget under the setup button.

THINGS TO TRY

Try using other R commands in the interactive shell, such as the statistical functions demonstrated in the other Simple R example models.
