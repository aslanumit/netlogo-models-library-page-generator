Example 1

Run on NetLogoWeb

WHAT IS IT?

This model demonstrates basic communication with R using the Simple R extension.

HOW IT WORKS

The model first demonstrates how to call an R function and retrieve the result with sr:run-result, using the R function rnorm as an example. Then, the model creates a set of turtles and assigns them to an R variable, using sr:set-agent. The rest of the procedure demonstrates various basic ways of manipulating data in an existing R variable.

HOW TO USE IT

Click the test button, and observe the contents of the output widget.

EXTENDING THE MODEL

Try using other R functions to manipulate the turtle data, such as computing the average xcor position of the turtles using the mean function.
