Image Resource Example

Run on NetLogoWeb

WHAT IS IT?

This code example demonstrates the use of bundled images. It is functionally equivalent to the Image Import Example model, but instead of reading the images from local files, the images are read from bundled resources and loaded with the Import-A extension.

HOW TO USE IT

Select the sample image you want to import with the FILENAME chooser, and then press any of the buttons below the chooser.

THINGS TO NOTICE

The drawing appears in front of the patch colors.

Images imported by import-a:pcolors and import-a:pcolors-rgb are scaled to fit the world. The colors when using import-a:pcolors may be distorted since NetLogo supports only a limited color set for pcolor values. When using import-a:pcolors-rgb the colors will be set to three-item RGB lists. See the Color section of the Programming Guide in the NetLogo User Manual for information on colors in NetLogo.

THINGS TO TRY

Try loading the different images into the patches to get an idea of the type of color distortion to expect.

Try importing an image with import-a:pcolors and then import the same image with import-a:drawing and import-a:pcolors-rgb. How does handling of the image differ?
